# Submitter Walkthrough Images

This folder contains the screenshots used in the submitter walkthrough documentation.

## Expected Image Files

Place the following screenshot files in this directory:

1. `submissions-dashboard-empty.png` - Empty submissions dashboard (no submissions yet)
2. `select-grant-program.png` - Step 1: Grant program selection interface
3. `fill-in-form.png` - Step 2: Application form with milestones
4. `submit-proposal.png` - Final submission step with save/submit buttons
5. `submissions-dashboard-with-submissions.png` - Dashboard showing submitted proposals
6. `pending-submission-status.png` - Detailed submission status page (pending/under review)

## Image Specifications

- Format: PNG preferred
- Naming: Use lowercase with hyphens (kebab-case)
- Size: Optimize images for web (recommended max width: 1200px)
