# Review Walkthrough Images

This folder contains the screenshots used in the review walkthrough documentation.

## Expected Image Files

Place the following screenshot files in this directory:

1. `grant-dashboard.png` - The main Grant Dashboard showing navigation and overview
2. `review-dashboard.png` - The Review Dashboard with pending actions
3. `cast-your-vote.png` - The vote casting dialog/modal
4. `submission-review.png` - Submission review status page with voting statistics
5. `all-milestones.png` - Overview of all milestones for a submission
6. `review-milestone.png` - Milestone review dialog with details

## Image Specifications

- Format: PNG preferred
- Naming: Use lowercase with hyphens (kebab-case)
- Size: Optimize images for web (recommended max width: 1200px)
